This is an example of the **splash** layout. It's perfect for:

- Landing pages
- Home pages
- Hero sections
- Marketing pages

## Features

- Full-width header with background image
- Overlay effects
- Call-to-action buttons
- No sidebar (full-width content)

## Front Matter Options

```yaml
layout: splash
title: "Splash Page Title"
permalink: /splash-url/
header:
  overlay_color: "#000"  # Hex color
  overlay_filter: "0.5"  # 0.0 to 1.0
  overlay_image: /path/to/image.jpg
  actions:
    - label: "Button 1"
      url: "/link1/"
    - label: "Button 2"
      url: "/link2/"
  caption: "Caption text"
excerpt: "Short description shown below header"
```

## Customization Tips

- Use high-quality images (1920x1080 or larger)
- Adjust `overlay_filter` to make text readable
- Add multiple action buttons for different CTAs
- Keep excerpt short and compelling
