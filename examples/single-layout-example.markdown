This is an example of the **single** layout. It's perfect for:

- Blog posts
- Individual pages
- Articles
- Documentation pages

## Features

- Clean, focused content area
- Optional header image with overlay
- Author profile sidebar (if `author_profile: true`)
- Table of contents (auto-generated for long posts)
- Social sharing buttons
- Related posts at the bottom

## Front Matter Options

```yaml
layout: single
title: "Your Page Title"
permalink: /your-url/
author_profile: true  # Shows author sidebar
header:
  image: /path/to/image.jpg
  overlay_image: /path/to/overlay.png
  overlay_filter: 0.5
  caption: "Caption text"
  actions:
    - label: "Button Text"
      url: "/link/"
tags: [tag1, tag2]  # For categorization
categories: category-name
```
