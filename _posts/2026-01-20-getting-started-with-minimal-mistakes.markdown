## What is Minimal Mistakes?

Minimal Mistakes is a flexible two-column Jekyll theme perfect for building personal sites, portfolios, documentation, and blogs.

## Key Features

1. **Multiple Layouts** - Choose from single, archive, splash, and more
2. **Responsive Design** - Works great on all devices
3. **Easy Customization** - Simple configuration options
4. **SEO Friendly** - Built with search engines in mind

## Using This Theme

To use Minimal Mistakes, you need to:

1. Set the theme in `_config.yml`
2. Choose the right layout for each page
3. Customize the navigation and author info
4. Add your content!
